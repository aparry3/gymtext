# Workout Message Examples

Reference examples for GymText workout messages. Each follows the [Message Format Standard](../docs/MESSAGE_FORMAT.md).

---

## 1. Strength Focus (Upper Body)

**Scenario:** Intermediate lifter, Day 1 of upper/lower split. Heavy compound movements with low reps.

```
Upper Body Strength

Warm-Up:
- Band pull-apart: 2x15
- Arm circles: 30s each direction

Workout:
- BB bench press: 5x5
- Weighted pull-up: 4x6
- Overhead press: 4x6
- DB row: 3x8 each side
- Dips: 3x8

Notes: Rest 2-3min between main lifts. Aim for RPE 8.
```

---

## 2. Hypertrophy Focus (Legs)

**Scenario:** Intermediate lifter, leg day. Higher volume, moderate weight, controlled tempo.

```
Leg Hypertrophy

Warm-Up:
- Bodyweight squat: 2x10
- Leg swing: 10 each side

Workout:
- BB back squat: 4x10-12
- Romanian deadlift: 4x10-12
- Leg press: 3x12-15
- Walking lunge: 3x12 each leg
- Leg curl: 3x12-15
- Calf raise: 4x15-20

Notes: Rest 60-90s between sets. Control the eccentric (3s down).
```

---

## 3. Supersets (Push Day)

**Scenario:** Intermediate lifter, time-efficient push session using supersets.

```
Push Day

Warm-Up:
- Push-up: 2x10
- Band dislocate: 2x10

Workout:
- BB bench press: 4x8
- SS1 Incline DB press: 3x10
- SS1 Cable fly: 3x12
- SS2 Lateral raise: 3x15
- SS2 Tricep pushdown: 3x12

Notes: No rest between superset pairs. Rest 60s after each pair.
```

---

## 4. Circuit Training (Full Body)

**Scenario:** Beginner, limited equipment, full-body circuit. 3 rounds through each circuit.

```
Full Body Circuit

Circuit 1 (3 rounds):
- C1 Goblet squat: 3x10
- C1 Push-up: 3x12
- C1 DB row: 3x10 each side
- C1 Plank: 3x30s

Circuit 2 (3 rounds):
- C2 Reverse lunge: 3x10 each leg
- C2 DB shoulder press: 3x10
- C2 Bicycle crunch: 3x15 each side

Notes: Rest 15s between exercises, 60s between rounds.
```

---

## 5. Steady-State Cardio

**Scenario:** Runner building aerobic base. Easy effort day.

```
Easy Cardio

Run:
- Easy jog: 35min @ conversational pace

Cooldown:
- Walk: 5min
- Quad stretch: 30s each side
- Hamstring stretch: 30s each side

Notes: Keep heart rate under 140bpm. If you can't hold a conversation, slow down.
```

---

## 6. HIIT / Intervals

**Scenario:** Intermediate, sprint interval session on a bike or rower.

```
HIIT Cardio

Warm-Up:
- Light cycling: 5min
- Build-up sprint: 3x10s

Intervals:
- All-out sprint: 8x 30s on/60s off

Cooldown:
- Easy cycling: 5min
- Foam roll quads: 2min

Notes: Give max effort on each sprint. Full recovery between rounds.
```

---

## 7. Active Recovery

**Scenario:** Day after a hard training session. Goal is movement and blood flow, not intensity.

```
Active Recovery

Movement:
- Walk or light bike: 20min
- Foam roll full body: 10min

Mobility:
- Hip 90/90 stretch: 60s each side
- Cat-cow: 10 reps
- Thread the needle: 8 each side
- Deep squat hold: 60s

Notes: Keep everything easy. Focus on breathing and positions.
```

---

## 8. Rest Day

**Scenario:** Complete rest day. User gets a brief check-in.

```
Rest Day

No workout today. Recovery is part of the program.

If you feel like moving:
- Walk: 20-30min
- Stretching: 10-15min
```

## 9. Hybrid Workout (Strength + Cardio)

**Scenario:** Intermediate lifter who also needs cardio. Lower body strength followed by conditioning.

```
Lower Body + Conditioning

Warm-Up:
- Jump rope: 2min
- Bodyweight squat: 2x10

Strength:
- BB front squat: 4x6
- BB hip thrust: 4x8

Conditioning:
- KB swing: 5x15
- Assault bike: 3x 60s on/90s off

Cooldown:
- Walk: 3min
- Pigeon stretch: 60s each side

Notes: Rest 2min between strength sets. Push hard on conditioning.
```

## 10. Bodyweight Only (Travel/Home)

**Scenario:** User is traveling, no equipment available.

```
Bodyweight Workout

Warm-Up:
- Jumping jack: 60s
- Inchworm: 5 reps

Workout:
- Push-up: 4x15
- Bulgarian split squat: 4x10 each leg
- Pike push-up: 3x10
- Single-leg glute bridge: 3x12 each side

Core:
- Plank: 3x45s
- Dead bug: 3x10 each side
- Mountain climber: 3x20

Notes: Rest 45s between sets. No equipment needed.
```

## 11. Pull Day with Supersets

**Scenario:** Intermediate lifter, back/bicep focused session.

```
Pull Day

Warm-Up:
- Dead hang: 2x20s
- Band face pull: 2x15

Workout:
- BB deadlift: 4x5
- Weighted chin-up: 4x6
- SS1 Seated cable row: 3x10
- SS1 Reverse fly: 3x15
- SS2 Hammer curl: 3x12
- SS2 BB curl: 3x10

Notes: Rest 2-3min after deadlifts and chin-ups. No rest within supersets.
```

---

## 12. Sport-Specific (Running Intervals)

**Scenario:** Runner doing a speed workout. Track session with structured intervals.

```
Speed Work

Warm-Up:
- Easy jog: 10min
- A-skip: 2x30m
- High knees: 2x30m

Intervals:
- 400m repeat: 6x @ 5K pace
- Recovery jog: 90s between reps

Cooldown:
- Easy jog: 10min
- Calf stretch: 30s each side

Notes: Hit consistent splits. Don't go out too fast on rep 1.
```
