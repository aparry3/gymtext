Evaluate the output across these dimensions:

1. Program Appropriateness (Weight: 0.30)
- Score 10: Perfect match — program clearly designed for this specific user
- Score 8-9: Good fit with minor mismatches (e.g., 5-day program for a user with 4-5 days available)
- Score 5-7: Acceptable but generic (e.g., standard 4-day upper/lower for everyone)
- Score 3-4: Poor fit (e.g., 6-day program for a user with 3 days available, or complex periodization for a novice)
- Score 0-2: Fundamentally inappropriate (e.g., powerlifting program for a runner, ignores major constraints)

What to evaluate:
- Training split matches user's available days (e.g., 4-day program for a user with 4 days/week)
- Complexity matches experience level (simple for novices, complex for advanced)
- Program type matches goals (hypertrophy focus for muscle gain, strength for powerlifting)
- Respects constraints (schedule, equipment, injuries)
- Philosophy explains WHY this program fits this user

2. Exercise Selection & Programming (Weight: 0.25)
- Score 10: Excellent selection, variety, and specificity
- Score 8-9: Good with minor issues (e.g., one exercise doesn't quite fit the focus)
- Score 5-7: Acceptable but with notable problems (e.g., missing a key movement pattern)
- Score 3-4: Poor selection (wrong exercises for the goal, ignores constraints)
- Score 0-2: Fundamentally wrong (e.g., prescribes barbell squats despite active knee injury)

What to evaluate:
- Exercises match the stated focus (e.g., "Pull Day" has pulling movements)
- Exercise variety is appropriate (not repeating the same pattern excessively)
- Sets, reps, and rest periods match the goal (hypertrophy vs. strength vs. conditioning)
- Respects injury notes (e.g., goblet squat instead of barbell squat if knee issues)
- Equipment constraints are honored (no exercises requiring unavailable equipment)
- Specificity level is appropriate (prescriptive for meet prep, flexible for general training)

3. Periodization & Progression (Weight: 0.25)
- Score 10: Perfect periodization, clear progression, logical phase structure
- Score 8-9: Good with minor issues (e.g., deload timing not specified)
- Score 5-7: Acceptable but vague (e.g., "add weight when you can" without specifics)
- Score 3-4: Poor progression strategy (e.g., no deload, unclear phase transitions)
- Score 0-2: No discernible periodization or nonsensical progression

What to evaluate:
- Phases are logical and appropriate for the goal (accumulation → intensification → realization for meet prep)
- Progression strategy is clear (how to add weight, reps, or volume week-to-week)
- Deload strategy is included (every 3-4 weeks for periodized programs)
- Phase transitions are explained (what triggers moving to the next phase)
- For non-periodized programs: steady-state or autoregulated approach is clear

4. Clarity & Communication (Weight: 0.10)
- Score 10: Crystal clear, professional, actionable
- Score 8-9: Clear with minor verbosity or formatting issues
- Score 5-7: Somewhat clear but missing key details or overly verbose
- Score 3-4: Confusing, hard to follow, missing key sections
- Score 0-2: Incomprehensible or nonsensical

What to evaluate:
- Program Philosophy is concise (3-5 sentences) and explains WHY/HOW
- Weekly Pattern is clearly structured (Day — Session Name, Focus, Volume)
- Progression Strategy is actionable and specific
- Phase Cycling explains how phases connect
- Professional tone, no fluff, scannable formatting

5. Personalization & Detail (Weight: 0.10)
- Score 10: Highly personalized, clear evidence of profile integration
- Score 8-9: Good personalization with minor generic elements
- Score 5-7: Some personalization but mostly template-like
- Score 3-4: Minimal personalization, generic program
- Score 0-2: Completely generic, ignores profile entirely

What to evaluate:
- Program reflects specific profile details (equipment, preferences, constraints)
- Exercise rationale is provided for non-obvious choices (injury prevention, weak point work)
- Includes user-specific notes (e.g., "Friday is always lightest before Saturday long run" for runners)
- Not a generic template — clearly tailored to this individual
- Modification History section is included (empty for new programs)
