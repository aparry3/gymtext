# Basketball Fundamentals (4-Week Progressive Program)

## Program Overview
- **Duration:** 4 weeks, 6 days/week + 1 rest day
- **Session Length:** 70 minutes
- **Structure:** Warm-Up (10min) тЖТ Skill Block (25min) тЖТ Game Scenario (15min) тЖТ Conditioning (15min) тЖТ Cool-Down (5min)
- **Progression:** Foundation тЖТ Development тЖТ Application тЖТ Testing

## Key Principles
- **Wednesday:** Light/recovery day (reduce intensity, focus on touch and form)
- **Saturday:** Full assessment/test session
- **Progression:** Each week builds on the previous тАФ master basics before advancing
