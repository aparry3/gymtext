# Basketball Fundamentals (4-Week Progressive Program)

## Program Overview
- **Duration:** 4 weeks, 6 days/week + 1 rest day
- **Session Length:** 70 minutes
- **Structure:** Warm-Up (10 min) → Skill Block (25 min) → Game Scenario (15 min) → Conditioning (15 min) → Cool-Down (5 min)
- **Progression:** Foundation → Development → Application → Testing

## Key Principles
- **Wednesday:** Light/recovery day (reduce intensity, focus on touch and form)
- **Saturday:** Full assessment/test session
- **Progression:** Each week builds on the previous — master basics before advancing
