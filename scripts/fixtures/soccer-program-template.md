# Soccer Fundamentals (4-Week Progressive Program)

## Program Overview
- **Duration:** 4 weeks, 6 days/week + 1 rest day
- **Session Length:** 70 minutes
- **Structure:** Warm-Up (10min) ‚Üí Technical Block (25min) ‚Üí Game Scenario (15min) ‚Üí Conditioning (15min) ‚Üí Cool-Down (5min)
- **Progression:** Foundation ‚Üí Development ‚Üí Application ‚Üí Testing

## Key Principles
- **Wednesday:** Light/recovery day (reduce intensity, focus on touch and form)
- **Saturday:** Full assessment/test session
- **Progression:** Each week builds on the previous ‚Äî master basics before advancing
